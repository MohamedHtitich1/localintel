# DHS Health Indicator Codes

Returns a named vector of DHS indicator IDs for maternal and child
health. Covers antenatal care, skilled birth attendance, vaccination,
postnatal care, and contraceptive use.

## Usage

``` r
dhs_health_codes()
```

## Value

Named character vector of DHS indicator IDs.

## Examples

``` r
if (FALSE) { # \dontrun{
codes <- dhs_health_codes()
names(codes)
} # }
```
