# DHS HIV/AIDS Indicator Codes

Returns a named vector of DHS indicator IDs for HIV/AIDS prevalence,
testing, knowledge, and treatment.

## Usage

``` r
dhs_hiv_codes()
```

## Value

Named character vector of DHS indicator IDs.

## Examples

``` r
if (FALSE) { # \dontrun{
codes <- dhs_hiv_codes()
names(codes)
} # }
```
