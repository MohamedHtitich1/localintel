# DHS Nutrition Indicator Codes

Returns a named vector of DHS indicator IDs for child and maternal
nutrition. Covers stunting, wasting, underweight, anemia, obesity, and
breastfeeding indicators.

## Usage

``` r
dhs_nutrition_codes()
```

## Value

Named character vector of DHS indicator IDs.

## Examples

``` r
if (FALSE) { # \dontrun{
codes <- dhs_nutrition_codes()
names(codes)
} # }
```
