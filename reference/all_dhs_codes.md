# All DHS Indicator Codes

Returns a comprehensive named vector combining indicator codes from all
8 DHS thematic domains. This is the full registry of DHS subnational
indicators that localintel can process.

## Usage

``` r
all_dhs_codes()
```

## Value

Named character vector of all DHS indicator IDs.

## Examples

``` r
if (FALSE) { # \dontrun{
all_codes <- all_dhs_codes()
length(all_codes)
} # }
```
