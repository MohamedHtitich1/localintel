# DHS Mortality Indicator Codes

Returns a named vector of DHS indicator IDs for child and infant
mortality. Covers under-5, infant, neonatal, and perinatal mortality
rates.

## Usage

``` r
dhs_mortality_codes()
```

## Value

Named character vector of DHS indicator IDs.

## Examples

``` r
if (FALSE) { # \dontrun{
codes <- dhs_mortality_codes()
names(codes)
} # }
```
