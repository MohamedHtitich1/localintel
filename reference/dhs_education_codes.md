# DHS Education Indicator Codes

Returns a named vector of DHS indicator IDs for education and literacy.
Covers literacy rates, school attendance, and educational attainment for
women and men.

## Usage

``` r
dhs_education_codes()
```

## Value

Named character vector of DHS indicator IDs.

## Examples

``` r
if (FALSE) { # \dontrun{
codes <- dhs_education_codes()
names(codes)
} # }
```
