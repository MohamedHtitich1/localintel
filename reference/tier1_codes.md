# Tier 1 DHS Country Codes

Returns a character vector of 15 Tier 1 validation countries for the SSA
expansion. These countries have 5+ survey rounds and represent
approximately 70% of SSA population.

## Usage

``` r
tier1_codes()
```

## Value

Character vector of 15 DHS country codes.

## Examples

``` r
if (FALSE) { # \dontrun{
tier1_codes()
} # }
```
