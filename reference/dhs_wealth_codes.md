# DHS Wealth and Asset Indicator Codes

Returns a named vector of DHS indicator IDs for household wealth and
asset ownership. Covers wealth quintile distribution and key asset
indicators.

## Usage

``` r
dhs_wealth_codes()
```

## Value

Named character vector of DHS indicator IDs.

## Examples

``` r
if (FALSE) { # \dontrun{
codes <- dhs_wealth_codes()
names(codes)
} # }
```
